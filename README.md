## Notes

- `Mihomo/lazy_group.public.yaml` is a token-free template. Replace `https://example.com/your-subscription-url` with your own subscription URL in the client, or keep the full private config in the private repository.
- `Mihomo/outlook-network.override.js` is the Clash Verge override build generated from the same upstream rules and local custom rules.
- `Rules/*.list` files are published here so `rule-providers` can be fetched through Raw/CDN URLs.
- Do not publish real proxy subscription tokens in this repository.
